# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-05

### Added
- Initial release of imrank
- PyQt5-based GUI for ranking images
- Support for .jpg files
- Keyboard shortcuts for navigation and ranking
- Resume functionality with -c/--continue flag
- Sortable rankings list with real-time updates
- Cross-platform support (macOS and Windows)
- pip installable package
